== Description ==

A plugin to create and manage popups on the homepage.

== Installation ==

1. Upload the `custom-popups` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a new popup by going to the Popups menu and adding a new post.
